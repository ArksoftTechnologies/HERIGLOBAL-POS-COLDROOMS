# Administrator Guide

## Initial Setup
1. **Login** as Super Admin.
2. **Create Outlets**: Go to Outlets management and create all physical locations.
3. **Create Users**: Create Outlet Admins and Sales Reps, assigning them to specific outlets.
4. **Create Categories**: Set up product categories.
5. **Add Products**: Import or create products with prices and reorder levels.

## Configuration
- **Payment Modes**: Enable/Disable payment methods (Settings).
- **Expense Categories**: Manage allow expense types.

## Daily/Weekly Tasks
- **Executive Dashboard**: Monitor company-wide performance.
- **Stock Transfers**: Approve transfers from Warehouse to Outlets.
- **Remittance Checks**: Review "Outstanding Remittances" report to ensure cash is being deposited.
- **Low Stock**: Check "Low Stock Alerts" and restock Warehouse/Outlets.

## Maintenance
- **Database Backup**: Regularly backup `heriglobal_pos.db` (or dump PostgreSQL DB).
- **User Audits**: Deactivate unused accounts.
